# Analyze_Intermediate_Images

## Overview
- Create ensemble point clouds(.spbr) from one input point cloud.

### Usage
```
$ ./analyzeIntermediateImages [input_file] [output_path]
```

### Example
```
$ ./analyzeIntermediateImages input.ply OUTPUT_DATA/LR10/
```
